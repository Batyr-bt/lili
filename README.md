# Мобильный сайт‑приглашение

Это простой статический сайт (одна страница), сделанный **под телефон**.

## Как запустить локально

### Вариант 1 (если установлен Python)
1. Открой терминал в папке проекта
2. Запусти:

`python -m http.server 5173`

Открой в браузере: `http://localhost:5173`

### Вариант 2 (если установлен Node.js)

`npx serve .`

## Как поменять текст (имена / дату / место)

Открой файл `script.js` и отредактируй объект `INVITE`:
- `toName` — кому приглашение
- `fromName` — от кого
- `when.isoLocal` — дата/время в формате `YYYY-MM-DDTHH:mm`
- `where` — место
- `dressCode` — дресс‑код
- `message` — текст

## Публикация

Можно залить как обычные статические файлы на любой хостинг (GitHub Pages, Netlify, Vercel, Tilda как HTML-блок и т.д.).

Примечание: используются шрифты Google Fonts (нужен интернет, либо можно заменить на системные).
